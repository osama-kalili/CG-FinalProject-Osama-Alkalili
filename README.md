Project Overview
Ancient Temple Escape is a 3D first-person exploration game developed using OpenGL (Core Profile 3.3).
The player navigates through an ancient temple filled with walls, corridors, and hidden artifacts.
The objective is to explore the environment, collect all the hidden relics, and reach the altar (exit) before the torch light runs out.
The project demonstrates key computer graphics concepts such as:
Lighting (ambient, diffuse, specular)
Texturing
Camera movement (first-person)
Basic collision detection
Fog effects and visual atmosphere

🎮 Gameplay Instructions
Objective
Collect all artifacts scattered around the map
Reach the altar to escape
Finish before time runs out
Controls
W → Move forward
S → Move backward
A → Move left
D → Move right
Mouse → Look around
ESC → Exit the game

⚙️ Features
First-person camera system
Real-time lighting with torch effect
Animated artifacts (floating and rotating)
Fog for atmospheric depth
Collision detection with walls
Timer-based challenge system
